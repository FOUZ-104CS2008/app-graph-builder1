
# App Graph Builder

This project is a Frontend Intern Task for **Ainyx Solutions**.  
It demonstrates an interactive canvas-based UI using ReactFlow.

## Tech Stack
- React + Vite
- TypeScript (strict)
- ReactFlow (xyflow)
- Zustand (state management)
- TanStack Query (mock data fetching)
- shadcn/ui
- Tailwind CSS

## Features
- App graph canvas with ReactFlow
- Three nodes with basic interactions
- Dotted background canvas
- Node selection using Zustand
- Right-side Node Inspector panel
- Tab-based inspector (Config / Runtime)
- Mock API integration using TanStack Query

## How to Run Locally
```bash
npm install
npm run dev
